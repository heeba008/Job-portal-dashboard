PROJECT OVERVIEW
================================================================================
Job Hub is a modern, interactive job portal dashboard built with vanilla HTML, 
CSS, and JavaScript. This single-page application simulates a complete job 
search and application platform, demonstrating advanced frontend development 
skills suitable for a beginner developer's portfolio.

The application provides a realistic job portal experience with user 
authentication, dynamic job listings, real-time search, application tracking, 
bookmark functionality, and a comprehensive dashboard interface.


CORE FEATURES
================================================================================

1. USER AUTHENTICATION SYSTEM
   - Login screen with username validation
   - Personalized user profile display
   - User avatar with initial letter
   - Session management with logout functionality
   - Welcome messages and user-specific greetings

2. DYNAMIC JOB LISTING SYSTEM
   - Pre-loaded with 6 diverse sample jobs
   - Job cards displaying:
     * Job title and company name
     * Location and employment type
     * Job description
     * Skill tags/requirements
     * Action buttons (Apply, View Details, Bookmark)
   - Visual status indicators for applied jobs
   - Smooth card hover animations

3. REAL-TIME SEARCH & FILTERING
   - Instant search across multiple fields:
     * Job titles
     * Company names
     * Locations
     * Skill tags
   - Live filtering as user types
   - Smart empty state display when no results found
   - Search query highlighting

4. JOB APPLICATION TRACKING
   - One-click apply functionality
   - Visual feedback showing applied status
   - Prevents duplicate applications
   - Application counter in user stats
   - Toast notifications for successful applications

5. BOOKMARK/SAVE FUNCTIONALITY
   - Star/unstar jobs for later viewing
   - Dedicated saved jobs sidebar panel
   - Live saved jobs counter
   - Click saved job to scroll to it in main grid
   - Visual indicators for bookmarked jobs

6. ADD NEW JOB POSTING
   - Collapsible form interface
   - Complete job submission form with fields:
     * Job Title (required)
     * Company Name (required)
     * Location (required)
     * Employment Type (dropdown: Full-time, Part-time, Contract, Remote)
     * Job Description (required)
     * Skills/Tags (comma-separated)
   - Form validation
   - Dynamic addition to job grid
   - Success confirmation messages

7. COMPREHENSIVE DASHBOARD STATISTICS
   - Real-time stat cards showing:
     * Total jobs in system
     * Applications sent by user
     * Saved jobs count
     * Available jobs remaining
   - Auto-updating counters
   - Visual gradient styling on numbers

8. SAVED JOBS SIDEBAR
   - Compact list view of bookmarked jobs
   - Shows job title and company
   - Click to navigate to job in main grid
   - Empty state when no jobs saved
   - Live counter badge

9. DARK/LIGHT MODE THEME TOGGLE
   - Beautiful dark mode implementation
   - Smooth color transitions
   - Persists preference in localStorage
   - Moon/sun icon toggle button
   - All UI elements theme-aware

10. TOAST NOTIFICATION SYSTEM
    - Success notifications (green)
    - Info notifications (blue)
    - Error notifications (red)
    - Slide-in animations
    - Auto-dismiss after 3 seconds
    - Contextual messages for all actions

11. SMART UI STATES
    - Empty search results state
    - No saved jobs state
    - Applied vs. available job states
    - Form validation messages
    - Loading and transition states

12. QUICK ACTION BUTTONS
    - View All Jobs (reset search)
    - Clear Search

    - Responsive button states

HOW TO USE
================================================================================

1. GETTING STARTED
   - Open job-portal-dashboard.html in any modern web browser
   - No server or build process required
   - Works completely offline

2. LOGIN
   - Enter any username on the login screen
   - Click "Login to Dashboard" to access the portal

3. BROWSING JOBS
   - Scroll through the job listings
   - Use the search bar to filter jobs
   - Click on tags to see similar positions

4. APPLYING TO JOBS
   - Click "Apply Now" on any job card
   - Track applied jobs in the statistics
   - Applied jobs show "✓ Applied" status

5. SAVING JOBS
   - Click the star icon to bookmark jobs
   - View saved jobs in the sidebar
   - Click saved jobs to navigate to them

6. ADDING NEW JOBS
   - Click "➕ Add New Job" button
   - Fill out the job posting form
   - Submit to add to the job grid

7. THEME SWITCHING
   - Click the moon/sun icon in header
   - Toggle between light and dark modes
   - Preference is saved automatically

8. LOGGING OUT
   - Click "Logout" button in header
   - Returns to login screen
   - Clears application state
